Built for developers who care about DX.

Drop in the SDK, initialize once, and call one method. Syncra handles background workers, retries, and conflict resolution automatically.

  • check_circleTypeScript native — full type safety
  • check_circleWorks with React, Vue, Svelte, or vanilla JS
  • check_circleEvent-driven — listen to sync, conflict, online/offline
  • check_circleZero config — works out of the box
syncra-init.ts 
import { SyncraSDK } from 'syncra-sdk';

// Initialize once — works offline immediately
const sdk = new SyncraSDK({
  baseUrl: 'https://api.syncra.dev',
  token: 'your-api-key',
  syncInterval: 30000,
});

await sdk.initialize();

// Create records — queued offline, synced when online
await sdk.createRecord({ title: 'My note' });

// Listen to sync events
sdk.on('sync-complete', ({ applied }) => {
  console.log(`Synced ${applied} records`);
});

// Custom conflict resolution
sdk.onConflict((conflict) => ({
  id: conflict.recordId,
  data: conflict.serverData,
  version: conflict.serverVersion,
}));
